# Node 客户端请求示例
### 1. 安装依赖
```bash
npm install request
```

### 2. 运行
```bash
node example.js
```
- 说明：
    - 发送的图像需转为 `base64` 字符串
    - 请求参数为 `{"data": IMAGE_STRING}`
