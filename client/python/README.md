# Python 客户端请求示例
### 1. 安装依赖
```bash
pip install opencv-python

或者

pip install -r requirements.txt
```

### 2. 运行
```bash
python example.py
```
- 说明：
    - 发送的图像需转为 `base64` 字符串
    - 请求参数为 `{"data": IMAGE_STRING}`
